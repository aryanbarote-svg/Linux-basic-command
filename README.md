# Linux Basic Commands Assignment

## Q1: Creating and Renaming Files
mkdir test_dir  
touch example.txt  
mv example.txt renamed_example.txt  

## Q2: Viewing File Contents
cat /etc/passwd  
head -n 5 /etc/passwd  
tail -n 5 /etc/passwd  

## Q3: Searching Text
grep root /etc/passwd  

## Q4: Zip and Unzip
zip -r test_dir.zip test_dir  
unzip test_dir.zip  

## Q5: Downloading Files

## Q6: File Permissions
